O **E-commerce Java** é um sistema backend desenvolvido para simular as operações de uma loja virtual. A aplicação gerencia o fluxo completo de um e-commerce, desde o cadastro de produtos no catálogo até o processamento de pedidos e gerenciamento de usuários. 

Este projeto foi construído para aplicar conceitos sólidos de Programação Orientada a Objetos (POO) e arquitetura de APIs RESTful.

---

## Tecnologias

As seguintes ferramentas e frameworks foram utilizados na construção do projeto:

- **[Java](https://www.oracle.com/java/)** (Linguagem principal)
- **[Spring Boot](https://spring.io/projects/spring-boot)** (Framework backend)
- **[Maven](https://maven.apache.org/)** (Gerenciador de dependências)

---

## Como Executar

### Pré-requisitos
Certifique-se de ter instalado em sua máquina o [Java JDK](https://www.oracle.com/java/technologies/downloads/) e o [Git](https://git-scm.com/). 

### Rodando a Aplicação

```bash
# Clone este repositório
$ git clone [https://github.com/DalgonaFox/EcommerceJava.git](https://github.com/DalgonaFox/EcommerceJava.git)

# Acesse a pasta do projeto no terminal
$ cd EcommerceJava

# Instale as dependências e compile o projeto
$ mvn clean install

# Execute a aplicação
$ mvn spring-boot:run

```
